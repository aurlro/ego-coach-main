# Changelog

Toutes les modifications notables apportées à ce projet seront documentées dans ce fichier.

Le format est basé sur [Keep a Changelog](https://keepachangelog.com/en/1.0.0/), et ce projet adhère à la [Gestion Sémantique de Version](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-18

### Ajouté

-   **Analyseur Manuel** : Un wizard en 4 étapes pour guider l'utilisateur dans l'analyse des situations de conflit.
-   **Analyseur IA** : Intégration d'un moteur d'analyse multimodal (Texte + Image) utilisant des heuristiques locales, avec une compatibilité prête pour l'API Gemini.
-   **Journal de Bord** : Une fonctionnalité de sauvegarde locale (`localStorage`) pour conserver et consulter l'historique des analyses.
-   **Dojo d'Entraînement** : Un système interactif avec des scénarios prédéfinis pour s'exercer à la communication.
-   **Tableau de Bord** : Des statistiques en temps réel sur la page d'accueil pour suivre les progrès.
-   **Thème Sombre** : Un support complet du mode nuit pour une meilleure expérience visuelle.
-   **Bouton d'Urgence** : Une aide immédiate pour gérer les situations de crise en temps réel.

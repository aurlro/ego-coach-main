# Tools Directory

This directory contains utility scripts for development and testing.

## Python Scripts

- **verify_ui.py** - UI verification script
- **verify_ux.py** - UX verification script  
- **verify_ux_retry.py** - UX verification retry script

## Usage

These scripts are used for automated testing and verification during development. They require Python 3 and may have dependencies listed in the project's requirements.

To run a verification script:
```bash
python3 tools/verify_ui.py
```
